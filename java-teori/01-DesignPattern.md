# Design Pattern #

```merupakan best practice untuk memecah mecah masalah dalam secara umum```

* Jenis jenis
 1. Creation Pattern <berkaitan dengan hal pembuatan object / instansiasi>
   1.a. Singleton Pattern
   1.b. Factory Pattern /Factory Method
   1.c. Abstract Factory Pattern
   1.d. Prototype Pattern
   1.e. Builder Pattern
   1.f. Object Pool Pattern

 2. Structural Pattern <pola ini berhubungan dengan restrukturisasi class, seperti penggunaan inheritance, composition dalam object besar ke objek kecil>
   a. Adapter Pattern
   b. Bridge Pattern
   c. Composite Pattern
   d. Decorator Pattern
   e. Facade Pattern
   f. Flyweight Pattern
   g. Proxy Pattern

3. Behavior Pattern <berkaitan tentang interaksi dan response antar object>
   a. Chain of Responsibility Pattern
   b. Command Pattern
   c. Interpreter Pattern
   d. Iterator Pattern
   e. Mediator Pattern
   f. Memento Pattern
   g. Observer Pattern
   h. State Pattern
   i. Strategy Pattern
   j. Template Pattern
   h. Visitor Pattern
   i. Null Object
